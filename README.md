# HeartDiseasePrediction using Machine Learning

This project predicts whether a person is likely to have heart disease based on various medical parameters using a Machine Learning model. It was built as a beginner ML project to understand the complete workflow of a classification problem.

### Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- Google Colab

### Model
- Logistic Regression

### Features
- Data preprocessing and analysis
- Model training and evaluation
- Heart disease prediction for new patient data
